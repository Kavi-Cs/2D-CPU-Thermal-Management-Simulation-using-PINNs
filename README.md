# 2D-CPU-Thermal-Management-Simulation-using-PINNs
🚀 2D CPU Thermal Management Simulation using PINNs
This repository contains a Physics-Informed Neural Network (PINN) implementation to simulate heat diffusion and thermal management on a 2D CPU surface. Unlike traditional numerical solvers, this model learns the underlying physics of heat transfer directly from the governing partial differential equations (PDEs).
💡 Overview
Thermal management is a critical factor in semiconductor and hardware engineering. This project demonstrates how AI can be used to predict temperature distribution over time without relying on pre-existing datasets. By embedding physical laws into the loss function, the neural network acts as a continuous solver for the Heat Equation.
